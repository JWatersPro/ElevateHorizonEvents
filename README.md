# MobileDev_Project_Jacinda_W
My Major project for Mobile Dev

App Name:	Elevate Horizon Events App
Author:	Jacinda Waters


Project Purpose:	Elevate Horizon is a community events management app that allows users to find local fitness and entertainment. It has  detailed information, register for available spots, and customize their experience with theme preferences and accessibility settings.


Core Features:	
•	Home screen with today's events and featured carousel and a pull-to-refresh
•	Searchable Events List with remote JSON + sample data 
•	Event Details with date, time, location, capacity, status and a Register button
•	Event Register form with validation and simulated API submission
•	Settings screen with dark/light theme toggle, font size adjustment (small/medium/large) and sound toggle
•	Theming with brand colours
•	Unified loading/error states 


Tech Stack:	[Current Technology List]
•	React Native + Expo
•	React Navigation v6 (BottomTabs + StackNavigator)
•	React Native Paper (Material Design 3 components)
•	@react-native-async-storage/async-storage (persistence)
•	@expo/vector-icons (Ionicons)
•	Remote JSON API + local sample data fallback


Project Status:	
Full UI/UX implementation with wireframe fidelity, global theming, navigation flows, error handling.
